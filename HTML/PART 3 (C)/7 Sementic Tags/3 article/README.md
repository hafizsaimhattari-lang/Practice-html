## 🎯 Topics Covered

### 1. **Article Tag Basics**
`<article>` tag un cheezon ke liye best hai jaise:
- Blog posts
- News articles
- Forum posts
- User comments

Ye batata hai ke iske andar ka content independent hai aur baqi page ke content par depend nahi karta.

```html
<article>
    <header>
        <h2>Blog Post Title</h2>
        <p>Author and Date</p>
    </header>
    
    <p>Main content of the blog post goes here...</p>
    
    <footer>
        <p>Tags: HTML, Web</p>
    </footer>
</article>
```

---

## 📝 Summary
Ek page mein multiple `<article>` tags use ho sakte hain (jaise ek blog page par bohat sari posts hoti hain). Har article ke andar apna `<header>` aur `<footer>` bhi ho sakta hai (jaisa ke code example mein dikhaya gaya hai). Ye practice aapko yahi sikhane ke liye design ki gayi hai.
