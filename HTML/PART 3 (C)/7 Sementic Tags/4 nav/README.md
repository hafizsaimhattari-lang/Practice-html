## 🎯 Topics Covered

### 1. **Nav Tag Basics**
`<nav>` tag search engines aur screen readers ko batata hai ke is section mein website ke important navigation links (jaise Home, About, Services, Contact) mojood hain. Ye `<div>` tag ka ek semantic option hai menus banane ke liye.

```html
<nav>
    <a href="/home">Home</a>
    <a href="/about">About Us</a>
    <a href="/contact">Contact</a>
</nav>
```

### 2. **Best Practices**
- Har chote bare `<a>` (anchor) tag ko `<nav>` mein rakhna zaroori nahi hai. Sirf **major block of navigation links** (jaise navbar) ke liye iska use kiya jata hai.
- Ek page par ek se zyada `<nav>` tags ho sakte hain (jaise ek header mein main menu aur dusra footer mein terms/privacy links ke liye).
