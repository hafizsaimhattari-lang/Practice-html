## 🎯 Topics Covered

### 1. **Header Tag Basics**
`<header>` ek semantic tag hai jo search engines aur screen readers ko page ka top/introductory part define karne mein madad karta hai. Ye `<div>` tag ka ek behtar, semantic alternative hai.

```html
<header>
    <h1>Website Title</h1>
    <p>Subtitle or intro</p>
</header>
```

---

## 📝 Summary
Header tag sirf page ke top par hi nahi, balke kisi `<article>` ya `<section>` ke top par bhi use ho sakta hai unka intro dene ke liye. Isme aam taur par logo, navigation ya main heading hoti hai.
