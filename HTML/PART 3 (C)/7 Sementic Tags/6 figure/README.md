## 🎯 Topics Covered

### 1. **Figure Tag Basics**
`<figure>` tag kisi bhi self-contained media (jaise image) ko wrap karta hai. Iska maqsad ye batana hai ke ye image/media ek alag self-contained element hai jo main text ke sath juda hua hai.

### 2. **Figcaption Tag**
`<figcaption>` (Figure Caption) hamesha `<figure>` tag ke andar use hota hai aur ye us image ya media ki description/title dene ka kaam karta hai. Ye search engines ko bhi asani se samajh aata hai ke image kis baare mein hai.

```html
<figure>
    <img src="image.jpg" alt="A nice picture">
    <figcaption>A nice picture of a sunset.</figcaption>
</figure>
```

---

## 📝 Summary
Pehle HTML4 mein hum `<div>` aur `<p>` tag use karte the images ko caption dene ke liye. HTML5 mein `<figure>` aur `<figcaption>` ke aane se ye kaam zyada semantic, accessible aur search engine friendly ho gaya hai.
