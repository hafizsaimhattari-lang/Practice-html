# 🏷️ 7 Sementic Tags

## 📚 Is Folder Ka Maqsad
Yeh folder **HTML5 Semantic Tags** ke baare mein hai. Semantic tags wo tags hote hain jo apne meaning/content ko clearly describe karte hain browser aur developer dono ke liye (jaise `<header>`, `<footer>`, `<article>`, `<nav>`, waghera).

---

## 📁 Folder Structure

```
7 Sementic Tags/
├── 1 header/         (Introductory content)
├── 2 footer/         (Closing content)
├── 3 article/        (Self-contained content)
├── 4 nav/            (Navigation links)
├── 5 aside/          (Sidebar or related content)
├── 6 figure/         (Images and media with captions)
└── README.md         (This file)
```

---

## 🎯 Har Tag Ka Purpose

### 📁 1. header
Webpage ya kisi section ka top/introductory part banan ke liye. Aam taur par is mein logo, main heading aur nav hoti hai.

### 📁 2. footer
Webpage ya section ka bottom part banane ke liye. Is mein copyright, author details aur contact links hote hain.

### 📁 3. article
Aisa content jo apne aap mein independent ho aur alag se distribute kiya ja sake (jaise blog post ya news article).

### 📁 4. nav
Website ki main navigation (menu) links ke block ko darshata hai.

### 📁 5. aside
Aisa content jo main article ya page se indirectly related ho, isey aksar sidebars ke liye use kiya jata hai.

### 📁 6. figure (and figcaption)
Kisi image, diagram ya media element ko uski caption ke sath combine karne ke liye use hote hain.

---

## 📝 Summary
HTML4 mein in sab kaamo ke liye `<div>` tag use hota tha (jaise `<div id="header">`). Lekin HTML5 mein semantic tags aane ki wajah se code parhna asaan ho gaya hai, aur Search Engine Optimization (SEO) aur Screen Readers (Accessibility) mein bohat madad milti hai.
