## 🎯 Topics Covered

### 1. **`<div>` Tag (Block-Level Element)**
`<div>` ek block-level element hai, yani ye hamesha ek nayi line se shuru hota hai aur page ki poori chaurai (width) le leta hai. Ye bade sections (header, main content, footer) ko group karne ke liye use hota hai.

```html
<div class="box">
    <h2>This is a block</h2>
    <p>It takes up the full width.</p>
</div>
```

### 2. **`<span>` Tag (Inline-Level Element)**
`<span>` ek inline element hai, yani ye nayi line se shuru nahi hota aur sirf utni hi jagah leta hai jitna iske andar ka content hota hai. Ye paragraph ke andar kisi khas text (word ya sentence) ko color karne ya style karne ke liye use hota hai.

```html
<p>This is a normal paragraph but <span style="color: red;">this text is red</span>.</p>
```

---

## 📝 Summary
- **Div:** Bade blocks aur layout ke hissay banane ke liye.
- **Span:** Chote text ya line ke andar styling karne ke liye.
In donon tags ka apna koi visual effect nahi hota jab tak in par CSS na lagayi jaye.
