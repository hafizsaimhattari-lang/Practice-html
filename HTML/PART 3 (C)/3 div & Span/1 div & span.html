Practice: Div and Span Tags

This is a <div> Element

A div element is a block-level element. It always starts on a new line and takes up the full width available.

We use it to group other HTML elements together.

Another <div> Element

This is a separate block from the one above.

Inside this paragraph, we have a <span> element. Notice how the span does not start on a new line? It only takes up as much width as necessary.

We use span to style or target specific parts of text inside a block.
