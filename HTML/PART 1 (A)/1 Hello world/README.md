## 💡 Mukhtas Concepts

### DOCTYPE
```html
<!DOCTYPE html>
```
Browser ko batata hai: "Yeh modern HTML5 document hai"

### Meta Tags
```html
<meta charset="UTF-8">        <!-- Character support -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<!-- Mobile friendly banane ke liye -->
```

### Heading Hierarchy
```html
<h1>Main Topic</h1>           <!-- Ek hi page par -->
<h2>Sub Topic</h2>            <!-- Multiple use ok -->
<h3>Details</h3>              <!-- Kitni zaroorat ho -->
```
