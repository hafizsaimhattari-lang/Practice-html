## 🎓 Concepts Used

### 1. **Proper HTML Structure**
```html
<!DOCTYPE html>
<html>
    <head>
        <title>Hello World</title>
    </head>
    <body>
        <!-- Content -->
    </body>
</html>
```

### 2. **Heading Tags**
Problem solve karte waqt headings ka proper use:
- h1 = Main heading
- h2 = Sub-sections
- h3 = Details

### 3. **Paragraph Tags**
Content ko properly structured paragraphs mein likha.

### 4. **Formatting**
- `<b>` bold text
- `<p>` paragraphs properly

## ✅ Question Mein Kya Manga Tha

**Typical Q1 Type:**
> "Ek HTML page banao jishmein Avengers characters ke naam aur description ho"

**Solution Approach:**
1. Proper boilerplate se start
2. Main heading (h1)
3. Characters ke for each ek h2 heading
4. Description paragraph
5. Proper structure maintain

**Implementation:**
```html
<h1>Avengers</h1>

<h2>Iron Man</h2>
<p>Also goes by the name Tony Stark</p>

<h2>Captain America</h2>
<p>Also goes by the name Steve Rogers</p>

<h2>Hulk</h2>
<p>Also goes by the name Bruce Banner</p>
```

## ❌ Improvements

### 1. **Better Semantic Tags**
```html
<!-- Abhi -->
<h2>Iron Man</h2>
<p>Description</p>

<!-- Better Hota -->
<article>
    <h2>Iron Man</h2>
    <p>Description</p>
</article>
```

### 2. **Comments Add Karna**
```html
<!-- Character section -->
<h2>Iron Man</h2>

<!-- Character description -->
<p>Details...</p>
```

### 3. **Better Formatting**
```html
<!-- Abhi -->
<b>Tony Stark</b>

<!-- Better -->
<strong>Tony Stark</strong>
```
