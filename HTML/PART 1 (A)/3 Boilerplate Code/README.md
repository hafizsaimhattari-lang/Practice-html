---

## 📝 Har Line Ka Matlab

### Line 1: DOCTYPE
```html
<!DOCTYPE html>
```
**Kya Hota Hai:** Browser ko batata hai ke modern HTML5 use kar raha hoon
**Kyun Zaroori:** Browser properly render kare

---

### Line 2: HTML Root Element
```html
<html lang="en">
```
**Kya Hota Hai:** Poora document is tag ke andar hota hai
**`lang="en"`:** Language English hai (SEO ke liye)
**Kyun Zaroori:** Document ka starting point

---

### Lines 3-6: Head Section
```html
<head> 
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
```

**`<head>`** = Metadata section
- Browser ko settings
- Search engines ko info
- User ko direkt nahi dikhta

**`<meta charset="UTF-8">`**
- Character encoding
- Urdu, English, sab support karega
- **Zaroori:** UTF-8 use karna

**`<meta name="viewport">`**
- Mobile responsive banane ke liye
- Screen width pe adjust hoga
- **Zaroori:** Mobile friendly

**`<title>`**
- Browser tab mein jo text dikhta hai
- Search results mein dikhta hai
- **Zaroori:** Meaningful title likho

---

### Lines 7-end: Body Section
```html
<body> 
    <h1>Heading</h1>
    <p>Content</p>
</body>
```

**`<body>`** = Visible content
- User ko jo dikhta hai
- Sabhi content yahan likha jata hai

## 💡 Different Boilerplate Variations

### Minimal Boilerplate
```html
<!DOCTYPE html>
<html>
<head>
    <title>Page</title>
</head>
<body>
    <!-- Content -->
</body>
</html>
```

### Complete Boilerplate
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Page description">
    <meta name="author" content="Your name">
    <title>Page Title</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- Content -->
    <script src="script.js"></script>
</body>
</html>
```

### SEO Optimized
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Detailed description">
    <meta name="keywords" content="keyword1, keyword2">
    <meta property="og:title" content="Title">
    <meta property="og:description" content="Description">
    <title>Title - Keyword | Site Name</title>
</head>
<body>
    <!-- Content -->
</body>
</html>
```
