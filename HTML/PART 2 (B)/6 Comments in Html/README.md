## 🎯 Concepts

### Single Line Comment
```html
<!-- Yeh ek single line comment hai -->
```

### Multi-line Comment
```html
<!-- 
    Yeh ek multi-line comment hai
    Yahan detailed explanation likha jata hai
    Multiple lines use kar sakte ho
-->
```

### Code Comments
```html
<!-- Header section -->
<header>
    <!-- Navigation menu -->
    <nav></nav>
</header>

<!-- Main content -->
<main>
    <!-- Article section -->
    <article></article>
</main>
```
