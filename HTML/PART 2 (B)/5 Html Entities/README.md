## 🎯 Concepts

### Reserved Characters
```html
&lt;      <!-- < (Less than) -->
&gt;      <!-- > (Greater than) -->
&amp;     <!-- & (Ampersand) -->
&quot;    <!-- " (Quotation mark) -->
&apos;    <!-- ' (Apostrophe) -->
```

### Space Entities
```html
&nbsp;    <!-- Non-breaking space -->
&emsp;    <!-- Em space -->
&ensp;    <!-- En space -->
&thinsp;  <!-- Thin space -->
```

### Symbols
```html
&copy;    <!-- © (Copyright) -->
&reg;     <!-- ® (Registered) -->
&trade;   <!-- ™ (Trademark) -->
&deg;     <!-- ° (Degree) -->
&hellip;  <!-- … (Ellipsis) -->
&bull;    <!-- • (Bullet) -->
```

### Currency
```html
&pound;   <!-- £ (Pound) -->
&euro;    <!-- € (Euro) -->
&yen;     <!-- ¥ (Yen) -->
&cent;    <!-- ¢ (Cent) -->
```

### Emojis
```html
&#x1F606; <!-- 😆 -->
&#x1F916; <!-- 🤖 -->
&#x1F3C6; <!-- 🏆 -->
```
