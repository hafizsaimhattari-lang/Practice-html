Yeh folder **HTML Meta Tags** ke baare mein hai. Meta tags browser aur search engines (SEO) ko website ke baare mein important information dete hain.

---

## 🎯 Topics Covered

### 1. **Charset Meta Tag**
Character encoding set karta hai. UTF-8 sab characters ko support karta hai.
```html
<meta charset="UTF-8">
```

### 2. **Viewport Meta Tag**
Website ko mobile aur sab devices par responsive banane ke liye.
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 3. **Description Meta Tag**
Website ka short summary deta hai (SEO ke liye bohat important).
```html
<meta name="description" content="Hello This is Basic Website of html5">
```

### 4. **Keywords Meta Tag**
Search engines ke liye relevant keywords provide karta hai.
```html
<meta name="keywords" content="Html5, Basic Html Webiste,">
```

---

## 📝 Summary
Meta tags humesha `<head>` tag ke andar likhe jate hain. Ye user ko page par direct nazar nahi aate, lekin backend (browser aur search engine) inka heavily use karte hain website ko samajhne aur rank karne ke liye.
