## 🎯 Tags Covered

### Basic Formatting
```html
<b>Bold text</b>              <!-- Visual bold -->
<strong>Important</strong>    <!-- Semantic strong -->

<i>Italic text</i>           <!-- Visual italic -->
<em>Emphasized</em>          <!-- Semantic emphasis -->

<u>Underlined</u>            <!-- Underline -->
<s>Strikethrough</s>         <!-- Line through -->
```

### Advanced Formatting
```html
<mark>Highlighted</mark>      <!-- Yellow highlight -->
<del>Deleted</del>           <!-- Deleted text -->
<ins>Inserted</ins>          <!-- Inserted text -->
<sub>Subscript</sub>         <!-- H₂O -->
<sup>Superscript</sup>       <!-- E=mc² -->
```

### Code Formatting
```html
<code>code snippet</code>     <!-- Inline code -->
<pre>Preformatted</pre>       <!-- Preserves spacing -->
<kbd>Ctrl+S</kbd>            <!-- Keyboard input -->
<samp>Sample output</samp>   <!-- Sample output -->
```

### Other
```html
<small>Small text</small>     <!-- Smaller -->
<abbr title="Full">Title</abbr> <!-- Abbreviation -->
<q>Quote</q>                 <!-- Inline quote -->
<blockquote>Block quote</blockquote> <!-- Block quote -->
```
